# Phishing Detection Web Application

This project is a phishing detection system developed using Python and Flask.

## Features
- Detects suspicious URLs and inputs
- Basic security validation checks
- Logs and stores detection results
- Web interface for user interaction

## Technologies Used
- Python
- Flask
- HTML (templates)

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run the application:
   python web_app.py
4. Open browser:
   http://127.0.0.1:5000

## Purpose
This project demonstrates cybersecurity knowledge, phishing detection techniques, and web application development for SOC and IT support roles.
